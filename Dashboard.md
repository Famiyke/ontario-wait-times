# Ontario Surgical Wait Times Dashboard

Live dashboard: https://public.tableau.com/app/profile/ikenna.nwogu/viz/OntarioSurgicalWaitTimesHealthAnalyticsDashboard/ONTARIOSURGICALWAITTIMES-HEALTHANALYTICSDASHBOARD

Built in Tableau Public using ontario_waittimes.csv.

## Views Included

| View | Description |
|------|-------------|
| KPI Row | % hospitals meeting 90% target, avg compliance rate, wait time change 2018 vs 2023 |
| Target Compliance Heat Map | Red/green matrix by hospital and procedure — identifies worst combinations at a glance |
| Hospital Compliance Ranking | All 15 hospitals ranked by avg compliance, coloured by urban vs rural |
| Median Wait Time Trend | 2018–2023 line chart for Hip Replacement, Knee Replacement, and MRI with COVID-19 spike annotation |
| Hospitals Below 90% Target | Filtered bar chart showing only the 11 hospitals failing the provincial standard |

## Key Visual Findings
- 4 urban hospitals (Toronto General, Sunnybrook, Ottawa Civic, Mount Sinai) average above 90% compliance
- All 5 northern/rural hospitals average below 65% compliance
- The compliance gap between best and worst hospital is 29 percentage points
- 2020 shows a clear COVID-19 spike across all procedures and regions
